ChatGPT로 엑셀 매크로 만드는 법 — 반복 작업을 버튼 하나로

엑셀 매크로는 함수와 다릅니다. 여러 단계를 거치는 반복 작업을 버튼 하나로 묶는 기능입니다. 그 자동화 코드를 ChatGPT가 대신 짜 줍니다.

왜 함수가 아니라 매크로인가

함수는 값 하나를 계산합니다. 매크로는 여러 단계를 한 번에 실행합니다. 매달 똑같이 하는 정리·취합·서식 작업이 대상입니다. 그 순서를 말로 주면 ChatGPT가 코드로 바꿉니다. 아래 틀을 복사해 쓰세요.

바로 쓰는 프롬프트

당신은 엑셀 VBA 매크로 전문가입니다.
[매번 반복하는 작업]을 버튼 하나로 자동화하고 싶습니다. 데이터는 [시트·열 구성]입니다.
원하는 결과는 [최종 결과]입니다.
VBA 코드와 실행 방법 3줄로 알려주세요.

이렇게 채워서 씁니다

대괄호를 실제 상황으로 바꾸면 이렇게 됩니다.

당신은 엑셀 VBA 매크로 전문가입니다.
매달 부서별 시트 12개를 하나의 '전체' 시트로 취합하는 작업을 버튼 하나로 자동화하고 싶습니다. 데이터는 각 시트가 A열 날짜, B열 항목, C열 금액으로 같은 구조입니다.
원하는 결과는 모든 시트의 내용을 '전체' 시트에 아래로 이어 붙이고 맨 앞에 시트 이름(부서)을 넣는 것입니다.
VBA 코드와 실행 방법 3줄로 알려주세요.

이 프롬프트의 핵심

  • 함수가 아니라 여러 단계 반복 작업을 자동화 대상으로 정합니다
  • 매크로가 할 작업 순서를 1·2·3으로 적어 줍니다
  • 시트·열 구성을 알려줘 엉뚱한 코드를 막습니다
  • 받은 코드는 반드시 복사본에서 먼저 시험합니다

왜 복사본에서 먼저 시험하나 — 매크로는 실행하면 되돌리기가 안 됩니다. 잘못된 코드는 원본 데이터를 바로 바꿉니다. 그래서 원본을 백업하고 복사본에서 먼저 돌려 봅니다.

더 나아가려면 — 좋은 프롬프트의 4요소

반복 작업을 코드로 바꾸는 것도 결국 프롬프트 뼈대입니다. 좋은 프롬프트에는 뼈대가 있습니다 — 역할·맥락·지시·출력형식. 한 번 익히면 어디에나 응용됩니다. 원리 전체와 프롬프트 10개를 무료로 공개해 두었습니다.

정리

  • 매크로는 함수와 달리 여러 단계를 자동화합니다
  • 작업 순서와 시트·열 구성을 말로 줍니다
  • 받은 코드는 복사본에서 먼저 시험합니다
  • 실행 전 원본을 백업해 되돌리기에 대비합니다

자주 묻는 질문

ChatGPT가 준 매크로를 바로 실행해도 되나요?

바로 실행하지 마세요. 원본 파일을 백업하고 복사본에서 먼저 돌려 보세요. 매크로는 실행하면 되돌리기가 어렵습니다.

매크로 코드가 오류가 나면 어떻게 하나요?

오류 메시지를 그대로 붙여넣고 물어보세요. 어떤 작업을 시켰는지와 시트 구성도 함께 주세요. 원인과 고친 코드를 같이 받습니다.

매크로가 제 데이터에 딱 맞게 나오나요?

한 번에 딱 맞지 않을 때도 있습니다. 시트 이름과 열 위치가 실제와 다르면 코드도 어긋납니다. 그 부분을 정확히 알려주고 다시 요청하세요.
